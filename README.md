# jumpdir

## A better completion function for cd

[![](https://img.shields.io/badge/github-Vanille--N/jumpdir-8da0cb?logo=github)](https://github.com/Vanille-N/jumpdir)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

`jumpdir` [![](https://meritbadge.herokuapp.com/jumpdir)](https://crates.io/crates/jumpdir)
[![API](https://docs.rs/jumpdir/badge.svg)](https://docs.rs/jumpdir)

This project draws inspiration from [`z`](https://github.com/rupa/z)

It aims to retain the main functionality of `z` all the while being more predictable and suitable for specifying relative paths.
